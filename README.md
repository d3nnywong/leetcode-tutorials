# LeetCode 解题思路教程

一个**零基础友好**的 LeetCode 题解教程站。每道题都从「人话」讲起，配可交互的算法动画，让你看见每一步在做什么。

- 技术栈：**Vite + 原生 JS + 原生 CSS**（零框架，每行都能读懂）
- 第一题：**1143. 最长公共子序列**（二维动态规划入门）

## 站点功能

- **全屏三栏布局（桌面）**：左侧导航栏 + 中间正文 + 右侧「本页目录」（滚动高亮当前小节）；正文行宽受控以保证可读，结构块占满整列，宽屏不再留大片空白。窄屏自动收成抽屉菜单（汉堡按钮 + 遮罩），无横向滚动。
- **中 / EN 切换**：题目标题、难度、标签与**官方原题题面**在中英间切换——中文取自 leetcode.cn、英文取自 leetcode.com 的 1143 原题（逐字）。界面 chrome 同步切换；深入讲解正文保持中文。选择记到本地（localStorage）。
- **题目搜索**：侧边栏实时过滤题号 / 标题 / 英文名 / 标签；按 <kbd>/</kbd> 快速聚焦搜索框，<kbd>Esc</kbd> 关闭抽屉。
- **学习笔记**：每页右下角浮动「📝 笔记」按钮，侧滑面板里随手记，自动保存到浏览器本地（刷新不丢），支持导出 `.md` 和两步确认清空。不同页面的笔记互不干扰。
- **交互式 DP 动画**：逐格高亮当前格与来源格，可上一步 / 下一步 / 自动播放，支持自定义字符串实时重算，结束时高亮 LCS 路径。

## 本地运行

```bash
pnpm install     # 安装依赖（只需一次）
pnpm dev         # 启动本地开发服务器，按提示打开 http://localhost:5173
```

其他命令：

```bash
pnpm build       # 打包到 dist/
pnpm preview     # 本地预览打包后的结果
```

## 怎么再加一道题

1. 在 `src/data/problems.js` 的数组里加一条记录；
2. 复制 `src/problems/1143-lcs/` 改成新题的内容；
3. 在 `vite.config.js` 的 `input` 里加一行指向新页面。

首页会自动把新题列出来。

## 1143 题解页讲了什么

1. 读懂题目
2. 子序列 vs 子串（最容易混的概念）
3. 暴力法为什么慢 → 引出动态规划
4. DP 三件套：状态 / 转移 / 边界
5. **交互式填表动画**：逐格高亮当前格与来源格，可上一步/下一步/自动播放，支持自定义字符串
6. Java 题解（二维 DP）+ 复杂度分析
7. 进阶：滚动数组把空间压到 O(n)
